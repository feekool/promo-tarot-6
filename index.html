Быстрый старт

Запуск академии за 4 шага

Инструкция создана для практиков, а не разработчиков. Чётко, по делу, без технического жаргона.

  1. 1

    Аккаунт

    Зарегистрируйтесь на github.com. Бесплатно, 2 минуты, email.

  2. 2

    Шаблон

    Перейдите по ссылке github.com/feekool/feekool.github.io → Use this template.

  3. 3

    Наполнение

    Загрузите уроки, задайте название, логотип, цвета. Создайте ветки форума под темы курса.

  4. 4

    Публикация

    Settings → Pages → Branch: main → Save. Через 1-2 минуты платформа доступна по ссылке. Принимайте учеников.
